# Summer Income Planner

## Overview

Summer Income Planner is an agent skill for creating practical 6-week summer income plans for students, freelancers, and side-hustle beginners.

It turns a user's skills, time, constraints, and goals into a structured short-term plan with weekly action steps, beginner-friendly offer ideas, simple trackers, and quality-control checklists. It is designed to be useful without promising earnings, clients, or business success.

## When to use

Use this skill when a user asks for:

- A 6-week summer income plan.
- Student-friendly side-hustle ideas.
- A beginner freelance starter roadmap.
- Offer ideas based on their skills and schedule.
- Weekly action steps and accountability trackers.
- Quality-control checklists for outreach, delivery, and reviews.

## What it creates

A typical output includes:

- A safety and assumptions note.
- An intake snapshot.
- A menu of possible offers.
- A recommended primary and backup offer.
- A 6-week roadmap.
- Weekly action steps.
- Outreach or customer-discovery scripts.
- Trackers for time, outreach, leads, actual income, expenses, and lessons.
- Quality-control checklists.

## Package contents

- `SKILL.md`: agent instructions, triggers, workflow, output format, guardrails, and quality checklist.
- `README.md`: human-readable package overview.
- `agents/openai.yaml`: display metadata and invocation policy.
- `references/style-guide.md`: wording rules, plan structure, offer scoring, tracker guidance, and examples.
- `assets/intake-form.md`: reusable intake form.
- `assets/plan-template.md`: reusable 6-week plan template.
- `assets/weekly-tracker.csv`: spreadsheet-ready tracker template.
- `assets/offer-scorecard.csv`: offer comparison template.
- `assets/quality-control-checklist.md`: reusable QC checklist.
- `scripts/example_helper.py`: deterministic helper for generating a blank 6-week tracker CSV.

## Safety stance

The skill frames income goals as planning targets. It does not guarantee earnings, clients, conversion rates, profit, or business success. It avoids unsafe, illegal, deceptive, spammy, age-restricted, and regulated work, and it does not replace professional legal, tax, financial, or accounting advice.
